# VerduritasSAPP

Aplicación móvil para gestionar y planificar cosechas de cultivos.

## Características

- **Gestión de Cosechas**: Registra diferentes tipos de cultivos (tomates, cebollas, lechugas, apio, choclo) con sus fechas de cosecha.
- **Cálculo Automático**: Calcula automáticamente la fecha de cosecha basada en los días de maduración de cada cultivo.
- **Sistema de Autenticación**: Login y registro de usuarios mediante email/contraseña o Google.
- **Gestión de Datos**: 
  - Agregar nuevas cosechas
  - Editar cosechas existentes
  - Eliminar cosechas individuales
  - Borrar todas las cosechas
  - Prevención de duplicados

## Tiempos de Cosecha
- Tomates: 80 días
- Cebollas: 120 días
- Lechugas: 60 días
- Apio: 85 días
- Choclo: 90 días
